# Perplexity Space Query — промпты для копирования

Агент **пишет промпты**. Пользователь **копирует → вставляет в Perplexity → вставляет ответ обратно** в чат агенту.

**Контекст:** запросы делаются из Cursor. Есть связь с Linux (WSL, ssh или локальный терминал).

---

## 1. Workflow (единственный)

1. Агент формирует промпт по формату ниже
2. Агент выдаёт блок **«📝 Скопируйте и вставьте в Perplexity»**
3. Пользователь: Ctrl+C → в Perplexity → Ctrl+V → Enter
4. Пользователь ждёт ответ Perplexity, копирует его целиком
5. Пользователь вставляет ответ в чат агенту
6. Агент обрабатывает ответ (сравнение, сводка, следующие шаги — по контексту)

---

## 2. Формат промпта (обязательно)

Каждый промпт ДОЛЖЕН содержать:

```
[Контекст: учитывай предыдущие сообщения в этой беседе и в этом пространстве. Ответь на русском.]

[Источники: опирайся ТОЛЬКО на официальную документацию и best practices (текущий год). Указывай версии и ссылки.]

[Формат: дай полный ответ сразу целиком в одном блоке, чтобы его можно было скопировать одним действием (Ctrl+A или выделение всего текста).]

[Вопрос пользователя]
```

**Сокращённый вариант:**
```
Контекст: учти предыдущие сообщения. Источники: только офф. доки и best practices. Формат: ответ целиком одним блоком для разового копирования. [вопрос]
```

**Если вопрос завязан на проекте** (DevOps, деплой, kubectl, docker, скрипты, конфиги, команды и т.п.) — в промпт добавлять:

```
[Среда: запрос из Cursor, есть Linux. Если в ответе есть команды или конфиги — оформи отдельный блок «Вставка для Linux»: готовые команды/фрагменты для копирования в терминал, без лишнего текста, чтобы можно было вставить одним действием.]
```

---

## 3. Выдача промпта пользователю

После формирования промпта агент **всегда** оформляет блок:

```
📝 Скопируйте и вставьте в Perplexity
---
[полный промпт]
---
Инструкция: Ctrl+C → Perplexity → Ctrl+V → Enter → дождитесь ответа → скопируйте ответ целиком → вставьте сюда в чат.
```

---

## 4. Окно для вставки (Linux) — обязательно

Когда вопрос завязан на проекте (команды, конфиги, скрипты, деплой и т.п.), агент **обязательно** выдаёт блок:

```
🖥️ Вставка для Linux
---
[команды или конфиги — готовые для Ctrl+C → вставка в терминал]
---
Скопируй блок выше и вставь в терминал Linux.
```

Блок формируется из ответа Perplexity (если там есть блок «Вставка для Linux») или извлекается/собирается агентом из команд и конфигов в ответе.

---

## 5. После ответа пользователя

- Агент получает текст ответа Perplexity от пользователя
- Обрабатывает по контексту: сравнение с другими источниками, сводка, конкретные шаги, построчный разбор — в зависимости от задачи
- **Обязательно** выдаёт блок «🖥️ Вставка для Linux», если вопрос проект-зависимый
